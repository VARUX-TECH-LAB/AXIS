# Pre-Generated Evidence

This evidence was generated from a clean local pilot run. It is provided for reviewers who do not want to run code locally.

It does not replace running the package. It helps inspect expected behavior, audit evidence, response shapes, and intent. Reviewers should treat it as sample evidence, not proof of their own local execution.

Trust paradox:

We are security engineers, so we do not pretend pre-generated evidence replaces independently running the system. This folder shows expected evidence shape and behavior. The stronger validation path is running the pilot in your own environment or a disposable VM.

## Where To Look

- Summary: `demo/pre-generated-evidence/pilot-v1/evidence-summary.md`
- Selected responses: `demo/pre-generated-evidence/pilot-v1/selected-responses/`
- Selected audit events: `demo/pre-generated-evidence/pilot-v1/selected-audit-events/`
- Verification outputs: `demo/pre-generated-evidence/pilot-v1/verification-output/`

## Signed Status

This pre-generated evidence is not cryptographically signed in this package version.

Do not treat unsigned pre-generated evidence as a substitute for local verification.
